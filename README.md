## 🗂️ Repository Structure
```bash

├── CTF-Event-Name/          # CTF event directory
│   ├── Category/            # Challenge category (Web, Crypto, Pwn, etc.)
│   │   ├── Challenge-Name/  # Individual challenge directory
│   │   │   ├── WriteUp.md   # Detailed solution documentation

└── ..
```
